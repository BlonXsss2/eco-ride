🌿 EcoRide - Tests de Responsive Design

📱 Breakpoints CSS utilises

Appareil Largeur Breakpoint CSS Adaptations
Mobile 375px max-width: 576px Navigation hamburger, colonnes empilees, polices reduites
Tablette 768px max-width: 768px Grille 2 colonnes, hero reduit, images adaptees
Desktop 1200px+ min-width: 992px Layout complet, 3 colonnes, navbar horizontale

🔧 Technologies Responsive utilisees

  • Bootstrap 5 Grid System - Grille flexible avec classes col-md-*, col-lg-*
  • Meta Viewport - <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Media Queries CSS - Breakpoints a 576px, 768px et 992px
  • Flexbox - Disposition flexible pour navbar et cartes
  • Images responsives - object-fit: cover et largeurs relatives
  • Boutons tactiles - Taille minimale 44x44px sur mobile (accessibilite)
  • Navigation hamburger - Menu collapse Bootstrap pour ecrans petits

📐 Adaptations par page

Page d'accueil (index.php)

  • Mobile (375px): Hero en colonne, formulaire sous le texte, cartes empilees, police reduite a 1.5rem
  • Tablette (768px): Hero en colonne, 2 cartes par ligne, image about reduite a 250px
  • Desktop (1200px): Hero en 2 colonnes, 3 cartes par ligne, image about 400px

Resultats de recherche (carpools.php)

  • Mobile: Filtres caches dans un accordeon, resultats en 1 colonne
  • Tablette: Filtres en sidebar retractable, resultats en 2 colonnes
  • Desktop: Sidebar filtres + 3 colonnes de resultats

Details covoiturage (carpool_details.php)

  • Mobile: Tout en 1 colonne, conducteur sous les infos, bouton pleine largeur
  • Tablette: Infos trajet + sidebar conducteur
  • Desktop: 2 colonnes (8+4), infos completes avec avis detailles
